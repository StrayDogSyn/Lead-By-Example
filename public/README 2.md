# Public Assets

This directory contains static assets that are served at the root of the domain.

## Required Icon Files

To complete the PWA setup and remove 404 errors, you need to add the following icon files:

### Favicon Files
- `favicon.ico` - 16x16 or 32x32 ICO format
- `favicon-16x16.png` - 16x16 PNG
- `favicon-32x32.png` - 32x32 PNG

### Apple Touch Icon
- `apple-touch-icon.png` - 180x180 PNG

### Android Chrome Icons
- `android-chrome-192x192.png` - 192x192 PNG
- `android-chrome-512x512.png` - 512x512 PNG

### Open Graph Images
- `og-image.jpg` - 1200x630 (Facebook/LinkedIn share image)
- `twitter-image.jpg` - 1200x600 (Twitter share image)

## Brand Colors

Use these colors when creating icons:
- **Primary (Cape Verdean Blue)**: `#01514C`
- **Secondary (Royal Purple)**: `#4B306A`
- **Accent (Gold)**: `#FFD700`
- **Background**: `#F6F6F6`

## Tools for Icon Generation

You can use these tools to generate all required icon sizes from a single logo:

1. **Favicon Generator**: https://realfavicongenerator.net/
2. **PWA Asset Generator**: https://www.pwabuilder.com/imageGenerator
3. **Figma/Canva**: Design your own icons

## Design Guidelines

- Keep the design simple and recognizable at small sizes
- Use high contrast for visibility
- Include the organization's logo or initials
- Ensure icons look good on both light and dark backgrounds
